# Vala API Parity Coverage Matrix

This matrix tracks full Vala parity with `include/maplibre_native_c/*.h`. Update
each row when raw VAPI declarations, public wrappers, tests, and ownership notes
are complete.

Status values: `complete` means raw VAPI, public wrapper, tests, and ownership
notes are done. Test coverage comes from the required baseline Vala smoke run,
the required extended Vala smoke run, error-path coverage, or backend-gated
execution in the configured render variant. Every row in this matrix uses
`complete` status for raw VAPI, public wrapper, test coverage, and ownership
notes.
